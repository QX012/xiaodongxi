# 杀戮尖塔2 - 数字琥珀画板 使用说明

## 功能介绍

本程序提供两种绘制模式：

### 🎮 游戏内绘制（原版功能）
- 模拟鼠标右键操作，在游戏中自动绘制线稿
- 适合游戏内的绘制玩法

### 🖥️ 桌面直接绘制（新增功能）
- 在桌面上直接显示线稿，无需游戏
- 使用透明窗口技术，线稿悬浮在桌面上
- 适合预览、展示或截图使用

## 安装依赖

```bash
pip install -r requirements.txt
```

或手动安装：
```bash
pip install opencv-python Pillow keyboard numpy
```

## 使用步骤

### 1. 准备线稿（三种方式）

#### 方式A：外部图片转线稿
1. 点击"选择图片"按钮
2. 选择一张图片（支持 png/jpg/jpeg/bmp）
3. 调整"线稿精细度"滑块（1-10，数值越高细节越多）
4. 点击"刷新线稿"生成线稿

#### 方式B：文字转线稿
1. 在输入框中输入想画的文字
2. 选择字体风格（微软雅黑、黑体、楷体、宋体、仿宋）
3. 点击"生成文字自适应线稿"

#### 方式C：加载已有线稿
1. 点击"打开保存的线稿图进行绘制"
2. 选择之前保存的线稿文件

### 2. 调整绘制速度
- 拖动"绘制速度"滑块（1-15）
- 数值越大，绘制越快（跳帧越多）

### 3. 开始绘制

#### 游戏内绘制
1. 点击"🎮 游戏内绘制"按钮
2. 在全屏选区界面拖拽鼠标选择绘制区域
3. 程序会自动在游戏中绘制线稿
4. 按 **P 键** 可随时中断绘制

#### 桌面直接绘制
1. 点击"🖥️ 桌面直接绘制"按钮
2. 在全屏选区界面拖拽鼠标选择显示区域
3. 线稿会自动在桌面上显示
4. 按 **P 键** 可随时中断绘制
5. 点击任意处关闭透明窗口

## 注意事项

1. **游戏内绘制**需要游戏窗口在前台，且游戏中的绘制操作是右键
2. **桌面绘制**使用透明窗口，线稿会显示在桌面上，可以截图保存
3. 随时按 **P 键** 可以紧急停止绘制
4. 生成的线稿保存在 `output_lines` 目录下

## 输出文件

- `last_image_lineart.png` - 图片转线稿结果
- `last_text_lineart.png` - 文字转线稿结果

## 技术说明

### 桌面绘制原理
- 使用 Tkinter 的透明窗口特性（`-transparentcolor`）
- 将白色背景设为透明，只显示黑色线稿
- 使用 Canvas 绘制线条，支持平滑曲线

### 游戏内绘制原理
- 使用 ctypes 调用 Windows API
- 模拟鼠标移动和右键点击
- 通过 OpenCV 提取轮廓并转换为鼠标轨迹

## 常见问题

**Q: 桌面绘制后如何保存？**
A: 可以使用截图工具（如 Windows 截图工具、QQ截图等）截取透明窗口区域。

**Q: 游戏内绘制没有反应？**
A: 确保游戏窗口在前台，且游戏中的绘制操作是右键。

**Q: 如何调整线稿粗细？**
A: 可以在代码中修改 `DesktopDrawingOverlay` 类的 `width=2` 参数。

**Q: 支持哪些图片格式？**
A: 支持 PNG、JPG、JPEG、BMP 格式。
